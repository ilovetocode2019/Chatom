Chatom

An open source messaging app.

Chatom is a quick, simple, and relibale messaging app, available for any device. Start messaging immediately, with no download required!


Web App


Open

Chatom is open source, which means you can see exactly how everything works or even modify the app.

Private

You're the only one with access to your conversations. If you want, you can even run your own messaging server.

Reliable

With a ... uptime in the past year, you can rely on Chatom to be available.

Consistent

No matter what device or platform you're using, you'll get the exact same experience.

Roadmap

Chatom is not finished yet, however it is in active development, which means that new features are very coming soon. Here are some features that are coming soon:
  • Grooup Chats
  • Enhancements like typing and attachments
  • Contacts
  • 2 Factor Authentication
